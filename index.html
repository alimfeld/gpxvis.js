gpxvis.js demo

JavaScript modules for visualizing GPX data.

This is a simple demo of gpxvis.js features. Select a sample GPX file and a track below. The track is displayed on the map (together with all the waypoints in the GPX file) and an elevation profile chart is created. A range control widget (just below the elevation profile chart) allows you to select the range of the track to display on the map and in the elevation profile chart.
